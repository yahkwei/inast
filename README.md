# Inast

Custom web and mobile application development — ghostly theme, mist animations, and order flows for Business and Individual clients.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Pages

| Route | Description |
|-------|-------------|
| `/` | Home — intro menu after splash |
| `/order` | Choose Business or Individual |
| `/order/business` | Full business scope form + document upload |
| `/order/individual` | Streamlined individual questionnaire |
| `/about` | About Us |
| `/contact` | Contact form |

## Form submissions

All contact and order forms send to **ghostarchives9@gmail.com** via [FormSubmit](https://formsubmit.co).

**First-time setup:** On the first form submission, FormSubmit will email `ghostarchives9@gmail.com` an activation link — click it once to start receiving submissions.

Submissions include the sender's email as the reply-to address so you can respond directly.
